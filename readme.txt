Add multiple Google Maps and place the page that you want with shortcode.


== Description ==

Easy to use can add multiple google maps.

**Custom Google Map Features**

* Custom marker
* Works in everywhere (sidebar,post,page,custom post)
* Fully customizable. Width,height,zoom,custom pin and map type
* Add location by coordinates

**Shortcode Example**
`[cgm id="5687"]`

== Installation ==

1. Upload `custom-google-map-cgm` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How can i add the new map? =

CGM Maps -> Add New

Fill the all fields set map icon from featured image section and Publish.

= How can i add my custom map on pages,post..etc. =

When you add new map, you can see shortcode. Example `[cgm id="4554"]`
Copy and paste the shortcode where you want.

== Screenshots ==

1. Go to CGM Maps on main Wordpress admin menu.
2. Click to Add New and fill what you change want.
3. Set map icon from Featured Image
4. And last, copy your shortcode and paste where you want.
